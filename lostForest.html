Lost Forest

A collective project to build a 3D playable environment. Assets and art created to work together to form a scene that can be used in a open world game. (Dec 2021)

Developed 2D/3D assets for this project such as mass rendering plants, trees and creating road and grass tilesets. Also managed the group's deadlines, updates and peer reviews while creating time for bug fixes when our code failed at times.
