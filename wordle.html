Wordle Clone

A wordle clone that is written in Java, HTML and CSS. (May 2022)

Wordle has been a game that took the world by storm and I've seen everybody even my mom playing this game everyday, so I thought to myself how would it look if I cloned the game myself. I researched for about a week on code, tutorials and programs needed to work on this project and found a few links that helped drastically in my production of this game. Something that changed my project was the use of CSS and its method of web page editing so that I could make it as similar to the genuine Wordle as possible.

You can find the link to my Wordle Clone here.

The primary meat of the code is based in script.js where it uses for loops to check if the 5 letter word inputted is correct by tracking every letter and matching it with the output of rightGuessString. The output rightGuessString takes a random answer from the dictionary of possible five letter words inside of words.js; this list contains all of the possible answers that a player can potentially guess. Once a player misses the answer 6 times, they lose and the right answer appears once the game ends. If I were to add anything else to this clone, it would be to add scoreboards and sharable scores to send to your friends, similar to the original Wordle.

Referenced from FreeCodeCamp.
